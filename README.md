Профессиональное desktop-приложение для ведения базы данных пациентов с автоматическим расчетом ИМТ и расширенной аналитикой. Разработано для медицинских учреждений и частных практик.

## Требования

### Минимальные системные требования:
- **Операционная система**: Windows 10/11, macOS 10.14+, Ubuntu 18.04+
- **Python**: версия 3.6 или выше
- **Оперативная память**: 4 ГБ RAM
- **Свободное место на диске**: 500 МБ

## Использование

### Для пользователей Windows:

1. **Скачайте Python** с [официального сайта](https://python.org/downloads)
2. **Установите Python** с галочкой "Add Python to PATH"
3. **Откройте командную строку** (Win + R → cmd)
4. **Выполните команды**:
