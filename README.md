## Требования

### Минимальные системные требования:
- **Операционная система**: Windows 7+, macOS 10.12+, или Linux (Ubuntu 16.04+)
- **Python**: версия 3.6 или выше
- **ОЗУ**: 512 МБ
- **Свободное место**: 10 МБ

### Поддерживаемые форматы изображений:
- PNG (рекомендуется)
- GIF

## Использование

### Для Windows пользователей:
1. Скачайте архив с программой
2. Распакуйте в любую папку
3. Запустите `calculator.py`

### Для продвинутых пользователей:
```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/tequila-calculator.git

# Перейдите в папку проекта
cd tequila-calculator

# Запустите калькулятор
python calculator.py
